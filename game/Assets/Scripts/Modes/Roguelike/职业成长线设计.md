## 一、通用系统概述

### 1.1 升级机制

- **经验曲线**: XP 基础 60，每级增长 ×1.15，最高 50 级
- **每次升级**: 从对应路线随机抽取 3 个可选升级项
- **三大路线**: `equipment`(装备)、`skill`(技能)、`player`(角色)
- **权重系统**:
  - Gameplay 类升级权重 ×1.35
  - Attribute 类升级权重 ×0.28
  - 已有标签每层叠加 +0.55 权重
  - 链式延续（满足前置）额外 ×2.4 权重

### 1.2 属性系统

所有职业属性通过 `RunBuildState` → `BuildStatRepository` 统一管理：
- 通用属性：生命、移速、暴击、吸血、减伤、全伤、经验加成等
- 职业专属属性：各职业有独立 StatKey 命名空间
- `Player` 路线升级对所有职业可用（生命、移速、暴击等通用属性）

### 1.3 升级路线分配

| 路线 | 法师 | 战士 | 投手 |
|------|------|------|------|
| `equipment` | ❌ | ✅ (`upgrades/warrior_progression.json`) | ✅ (`upgrades/ranged_upgrades.json`) |
| `skill` | ✅ (`mage_upgrades.json`) | ❌ | ❌ (Arena) / 可看到 (Explore) |
| `player` | ✅ (通用) | ✅ (通用) | ✅ (通用) |

## 二、法师 (Mage)

### 2.1 主题基础属性

| 属性 | 值 | 说明 |
|------|-----|------|
| 攻击倍率 | **0.9x** | 自动攻击较弱 |
| 攻速倍率 | **1.05x** | 攻速略快 |
| 基础射程 | **14.0** | 远距离施法 |
| 起始加成 | **技能伤害 +10%** | `skill_damage_mult +0.1` |

### 2.2 激活条件

```csharp
// 满足任一条件即激活 Mage Archetype:
WeaponTheme == "mage"
|| SkillTier > 0  // 已选过技能升级
|| SkillDamageMult > 1.01  // 技能伤害倍率 > 1.01
|| SkillCooldownReduce > 0  // 有技能冷却缩减
```

### 2.3 四大主动技能

| 槽位 | 技能 ID | 名称 | 类型 | 冷却 | 半径 | 持续时间 | 说明 |
|------|---------|------|------|------|------|----------|------|
| 1 | `skill_mage_arcane_bolt` | **奥术飞弹** | attack_profile | 1.2s | - | - | 基础投射物，短冷却主力输出 |
| 2 | `skill_mage_fire_nova` | **烈焰新星** | fire_nova | 3.5s | 2.8 | - | 范围爆炸，可附加灼烧 |
| 3 | `skill_mage_gravity_well` | **引力井** | gravity_well | 7.0s | 3.2 | 1.6s | 持续AOE + 黑洞吸引 |
| 4 | `skill_mage_frost_ward` | **寒霜结界** | frost_ward | 9.0s | 3.0 | 3.5s | 治疗 12%HP + 护盾 + 减速 + 击退 |

### 2.4 四大升级流派

法师拥有 **4 条技能升级树**，每条约 20 个节点，通过 `skill` 路线获得。所有升级带 `classes: ["mage"]` 标签。

- `skill_element_overload` — 超载反应
- `skill_element_burst` — 元素爆发
- `skill_chain_count` / `skill_chain_damage_ratio` — 连锁
- `skill_explosion_radius` — 爆炸半径
- `skill_burst_radius_bonus` — 爆发半径加成

- 终极形态实现"无限技能循环"

**关键 StatKey**:

- `skill_cd_reset_chance` — 冷却刷新概率
- `skill_time_stop_chance` — 时停概率
- `skill_time_rewind` — 时间回溯
- `skill_time_dilation_field` — 时停领域

### 2.5 六大被动行为 (Mage Behaviors)

Mage 拥有 6 个注册在 `MageBehaviorRegistry` 中的被动行为：

| 行为 ID | 类名 | 激活条件 | 效果 |
|---------|------|----------|------|
| `cooldown_reset` | CooldownResetBehavior | `SkillCdResetChance > 0` | 击杀时概率重置冷却 |
| `element_burst` | ElementBurstBehavior | `SkillElementBurst > 0` | 三元素同时触发大爆炸 |
| `element_melt` | ElementMeltBehavior | `SkillElementMelt > 0` | 冻结+火焰=蒸发伤害 |
| `element_overload` | ElementOverloadBehavior | `SkillElementOverload > 0` | 感电+雷电=连锁扩散 |
| `time_dilation` | TimeDilationFieldBehavior | `SkillTimeDilationField > 0` | 自身周围减速领域 |
| `time_stop` | TimeStopBehavior | `SkillTimeStopChance > 0` | 施法时概率时停 |

### 2.6 法术区域系统 (MageZone)

- `MageZone` 是可池化的持续伤害区域组件（每 0.25s tick）
- 支持的效果：
  - **持续拉取 (TickPull)**: 真空力 + 递增伤害 + 百分比血量伤害 + 易伤
  - **坍缩爆炸 (Collapse)**: 区域结束时的爆炸伤害
- 引力井 (Gravity Well) 技能利用此系统生成持久黑洞区域

### 2.7 法师成长总结

| 维度 | 特点 |
|------|------|
| **输出方式** | 4个主动技能 + 被动行为 |
| **核心属性** | 技能伤害倍率、技能范围、冷却缩减 |
| **成长方向** | 4条流派独立可选，混合搭配 |
| **弹体增殖** | 回响 + 镜像 + 额外弹体 + 命中分裂 |
| **元素系统** | 火(灼烧) + 冰(减速) + 雷(连锁) + 元素反应 |
| **控场能力** | 黑洞吸引、时间停止、减速、击退 |
| **生存能力** | 寒霜结界(治疗+护盾)、时停、减速 |
| **终极形态** | 每条流派 T20 为"天灾"级终极形态 |

## 三、战士 (Warrior)

### 3.1 主题基础属性

| 属性 | 值 | 说明 |
|------|-----|------|
| 攻击倍率 | **1.0x** | 标准 |
| 攻速倍率 | **1.0x** | 标准 |
| 基础射程 | **1.5** | 近战 |
| 攻击配置 | `weapon_theme_melee` | 近战攻击模板 |

### 3.2 激活条件

```csharp
// 仅当武器主题为 warrior 时激活
WeaponTheme == "warrior"
```

激活后禁用普通自动攻击，启用战刃星环被动。

### 3.3 核心机制：战刃星环 + 飞刃

战士是一个**被动环绕武器 + 自动投掷**职业：

| 机制 | 说明 |
|------|------|
| **战刃星环 (Orbit Blades)** | 3个球体环绕玩家旋转，接触敌人造成物理伤害 |
| **飞刃投掷 (Flying Blades)** | 自动向最近敌人发射飞刃（需要 `ExtraProjectile >= 1`） |
| **轨道流血 (Orbit Bleed)** | 星环命中施加流血 DoT |
| **投射物反射 (Deflect)** | 概率反弹敌方投射物 |

**战刃星环基础参数**:
- 武器数量: **3** 个球体
- 旋转速度: **205°/秒**
- 旋转半径: **1.85**
- 基础伤害: **10.0**
- 武器大小: **0.72**
- 命中冷却: **0.35s**

**飞刃基础参数**:
- 发射间隔: **1.2s**
- 飞刃速度: **12**
- 存活时间: **1.4s**
- 命中半径: **0.4**
- 弹跳次数: **0**（需升级解锁）

### 3.4 14 个专属升级

战士升级通过 `upgrades/warrior_progression.json` 定义，走 `equipment` 路线：

#### 升级树结构

```
blade_storm (星环增殖)           ─ 武器数量 +1
├── whirlwind (旋风)             ─ 旋转速度 +36
├── giant_orbit (外环拓展)       ─ 半径 +0.35
```

### 3.5 通用近战属性加成

战士的 `attack_profile_id` 为 `weapon_theme_melee`，以下通用近战属性也会生效：

- `melee_explosion_radius` — 近战爆炸半径
- `melee_knockback_chance` — 近战击退概率
- `melee_slow_chance` / `melee_slow_amount` — 近战减速
- `melee_burn_dps` / `melee_burn_duration` — 近战灼烧

## 四、投手 (Ranged/射手)

### 4.1 主题基础属性

| 属性 | 值 | 说明 |
|------|-----|------|
| 攻击倍率 | **1.0x** | 标准 |
| 攻速倍率 | **1.0x** | 标准 |
| 基础射程 | **16.0** | **全职业最远** |
| 起始加成 | **弱追踪** | `projectile_weak_homing = 1.0` |

### 4.2 激活条件

```csharp
// 满足任一条件即激活 Ranged Archetype:
WeaponTheme == "ranged"
|| ProjectileWeakHoming > 0
|| WeaponExtraProjectiles > 0
|| ProjectilePierce > 0
|| ProjectileTrailSpray > 0
|| LongRangeDamageMult > 0
```

### 4.3 核心机制：投射物行为系统

射手是一个**远距离弹幕输出**职业，核心是 5 种弹体行为：

| 行为 ID | 类名 | 激活条件 | 效果 |
|---------|------|----------|------|
| `pierce` | PierceBehavior | `ProjectilePierce > 0` | 弹体穿透敌人 |
| `split` | SplitBehavior | `ProjectileSplitOnHit > 0` | 命中时分裂出新弹体 |
| `chain` | ChainBehavior | `ProjectileChainCount > 0` | 弹体在敌人间连锁弹跳 |
| `homing` | HomingBehavior | `WeakHoming > 0.5 或 Homing > 0` | 弹体追踪敌人 |
| `explosion` | ExplosionBehavior | `ExplosionRadius > 0` | 弹体命中爆炸 |

### 4.4 五大升级流派

射手拥有 **5 条升级树**，每条约 20 级（共 100 个升级），通过 `equipment` 路线获得。所有升级带 `classes: ["ranged"]`。

---

#### 流派一：弹幕流 (Barrage) — `eq_ranged_bh_*`

**标签**: `projectile`, `rapid`, `spread`
**主题**: 攻速 → 额外弹体 → 尾迹喷洒 → 侧向散射 → 满屏弹幕

**核心属性**: `weapon_attack_speed_mult`, `weapon_extra_projectile`, `projectile_trail_spray`, `projectile_side_shed`, `projectile_split_on_hit`

**T20 终极**: **天灾弹幕** — 弹体 +2，尾迹 +2，散射，攻速 +25%，分裂 +1

**上限控制**: `weapon_extra_projectile < 8`, `projectile_side_shed < 2`

---

#### 流派二：狙击流 (Sniper) — `eq_ranged_sn_*`

**标签**: `projectile`, `critical`, `pierce`
**主题**: 暴击率 → 暴击伤害 → 精英/Boss 特攻 → 远程距离伤害 → 单发狙杀

**核心属性**: `crit_chance`, `crit_damage_mult`, `elite_damage_mult`, `boss_damage_mult`, `long_range_damage_mult`, `weapon_damage_mult`

**T20 终极**: **死神狙击** — 暴击率 +15%，暴伤 +80%，Boss/精英特攻

---

#### 流派三：爆炸流 (Explosion) — `eq_ranged_ex_*`

**标签**: `projectile`, `explosion`
**主题**: 爆炸半径 → 爆炸伤害比 → 黑洞吸引 → 全屏核爆

**核心属性**: `projectile_explosion_radius`, `projectile_explosion_ratio`, `explosion_damage_mult`, `explosion_vacuum`

**特色机制**: T11 解锁黑洞吸引（`explosion_vacuum`），爆炸时拉拽敌人

**T20 终极**: **灭世核爆** — 超大爆炸 + 黑洞 + 二次爆炸余波

**上限控制**: `explosion_vacuum < 1`

---

#### 流派四：冰冻流 (Freeze) — `eq_ranged_fr_*`

**标签**: `projectile`, `ice`
**主题**: 减速概率 → 减速量 → 脆冰增伤 → 冻结斩杀

**核心属性**: `projectile_slow_chance`, `projectile_slow_amount`, `slow_target_damage_mult`

**核心机制**: 高减速控制 + 对减速目标伤害倍率（"脆冰"机制）

**T20 终极**: **冰河世纪** — 减速概率 +35%，减速量 +40%，脆冰倍率 +45%

---

#### 流派五：雷电流 (Lightning) — `eq_ranged_lt_*`

**标签**: `projectile`, `lightning`, `chain`
**主题**: 连锁次数 → 连锁伤害比 → 感电灼烧 → 连锁清屏

**核心属性**: `projectile_chain_count`, `projectile_chain_damage_ratio`, `projectile_burn_dps`, `projectile_burn_duration`

**T20 终极**: **雷神降临** — 连锁 +3，连锁伤害 +25%，感电轰击

**上限控制**: `projectile_chain_count < 10`

## 五、三职业对比总结

### 5.1 核心差异

| 维度 | 法师 (Mage) | 战士 (Warrior) | 投手 (Ranged) |
|------|------------|---------------|--------------|
| **输出方式** | 4 主动技能 | 被动环绕 + 自动飞刃 | 远距离投射物 |
| **操作复杂度** | 高（管理4个技能冷却） | 低（专注走位） | 中（自动攻击 + 走位） |
| **升级路线** | `skill` 路线 | `equipment` 路线 | `equipment` 路线 |
| **升级流派** | 4 条（奥术/元素/引力/时序） | 1 条（14节点树） | 5 条（弹幕/狙击/爆炸/冰冻/雷电） |
| **升级总数** | ~80 个 | 14 个 | 100 个 |
| **基础射程** | 14.0 | 1.5 | **16.0** |
| **起始加成** | 技能伤害 +10% | — | 弱追踪 |
| **弹体行为** | 回响/镜像/分裂/追踪/连锁 | 星环/飞刃/反射/弹跳 | 穿透/分裂/连锁/追踪/爆炸 |
| **控场能力** | 黑洞/时停/减速/击退 | 流血 | 减速/黑洞吸引/感电 |
| **生存能力** | 寒霜结界(治疗+护盾) | 投射物反射 | 减速控场 |
| **终极形态** | 各流派 T20 "天灾" | 满 14 升级即满 | 各流派 T20 "传奇" |

### 5.2 共同系统

- **通用属性升级**: 三职业均可通过 `player` 路线获得生命、移速、暴击、吸血、减伤等通用属性
- **经验/等级系统**: 统一 XP 曲线（60 基础，×1.15 增长，50 级上限）
- **升级选择**: 每级 3 选 1，权重系统统一
- **属性存储**: 统一 `BuildStatRepository` → `RunBuildState` Facade
- **标签系统**: 标签堆叠影响升级权重和链式延续

### 5.3 设计哲学

| 职业 | 设计理念 |
|------|----------|
| **法师** | "技能组合大师" — 4条流派可混合搭配，通过元素反应、引力控场、时间操控实现多样化 Build |
| **战士** | "被动杀戮机器" — 精简但高效的 14 节点树，专注半径扩张和流血叠加，适合休闲玩家 |
| **投手** | "弹幕工程师" — 5条流派覆盖从弹幕海到狙击再到核爆的完整投射物体验，最多选择空间 |
